Amazon Image Size Requirements

Amazon requires product images to be at least 1000px on the longest side to enable zoom functionality. Higher resolution images improve customer confidence and conversions.

The recommended size for Amazon product photos is 2000×2000 pixels, which ensures compatibility across all listings. Using JPEG format is preferred for a balance of quality and file size.
